Cratle Customs

This module allows users to submit import and export declarations to UK customs. They're checked by a Cratle Customs agent before being submitted.

Whilst your first declaration is free, Cratle make a charge for each declaration after that. The money is used to reinvest in the module, pay agents, and to pay the port authorities for access to their systems.

Installation

There aren't any special instructions to install the module.

Configuration

To configure this module, you need to:

  • Add your Administrator to Customs Declarations Administrators group
  • Designate a user to be the 'Data Administrator' and add them into the Data Changes group
  • Discuss the documents necessary to support your declarations with Cratle Customs by emailing us at <odoo@cratle.co.uk>
  • Configure the documents, either through the Settings menu or by reloading the module with updated configuration files.

Usage

Customs declarations are split into headers and items. The header contains information common to all elements of the declaration such as the port of arrival and the transport method and the items contain information about the goods that are being moved across the border.

Both headers and items can be created in the module, headers are termed 'Declarations' and items are 'Items'.

To use this module, you need to:

  1. Create a new declaration.
  2. Create one or more items and attach them to the declaration.
  3. Lock the declaration against further changes.
  4. Send the email to customs. That will send the declarations to Cratle Customs to be checked before it is forwarded to HMRC.

For further information, please visit:

  • Cratle help on the web

Known issues / Roadmap

  • Until the HMRC CDS system is generally available, declarations cannot be sent to HMRC directly without being routed through a Customs Agent.
  • This is an intentionally simplified interface to create a customs declaration. If you have more complex requirements, Cratle Customs can fulfil those too.
